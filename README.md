# AI秘書パッケージ

**経営層・管理職のための AI 秘書室を、即日セットアップ。**

Claude Code を使って、あなたの組織に AI 秘書を導入するパッケージです。
タスク管理ツールではなく、**意思決定を速くするための思考パートナー**として設計しています。

---

## こんな方に

- 1人または少人数で事業を動かしている経営者・管理職
- 情報が散らばって、判断が遅くなっていると感じている
- 会議・商談・タスクの記録を仕組みとして残したい
- AIを使って業務を効率化したいが、何から始めればいいかわからない

---

## 特徴

### 秘書室だけから始められる

まず軽く試す。合ったら部署を追加する。それだけです。

### 経営の言語で動く

TODO管理ツールではありません。
論点の整理・選択肢の提示・意思決定の記録に特化しています。

### 必要な部署を後から追加

事業フェーズに合わせて、営業・マーケ・PM などを拡張できます。

---

## インストール

```bash
# 1. このリポジトリをクローン
git clone https://github.com/kyoueyama/ai-secretary-package.git

# 2. 使いたいプロジェクトのディレクトリに移動
cd your-project

# 3. インストール実行
bash path/to/ai-secretary-package/install.sh
```

インストール後、Claude Code を起動して話しかけるだけです。

```
「今日の優先課題を整理して」
「〇〇について壁打ちしたい」
「意思決定を記録しておいて」
「ダッシュボードを見せて」
```

---

## 部署の追加

後から部署を追加したいときは：

```bash
bash path/to/ai-secretary-package/add-department.sh
```

| 部署 | 用途 |
|------|------|
| 営業 | クライアント管理・提案書・商談履歴 |
| マーケティング | コンテンツ企画・SNS戦略・キャンペーン |
| PM | プロジェクト進捗・マイルストーン管理 |
| 経理 | 請求書・経費・売上管理 |
| リサーチ | 市場調査・競合分析 |
| 開発 | 技術ドキュメント・設計書・デバッグログ |
| クリエイティブ | デザインブリーフ・ブランド管理 |
| 人事 | 採用管理・チーム管理 |

---

## ファイル構成

インストール後、以下の構成が作成されます：

```
.secretary/
├── CLAUDE.md              ← 組織ルール
└── secretary/
    ├── CLAUDE.md          ← 秘書の定義
    ├── inbox/             ← クイックキャプチャ
    ├── todos/             ← 日次タスク
    └── notes/             ← 意思決定ログ・会議録
```
