# AI秘書パッケージ

経営層・管理職のための AI 秘書導入パッケージです。
Claude Code を使って、あなたの組織にAI秘書室を即日セットアップします。

## コンセプト

- **秘書室だけからスタート** — まず軽く始める
- **必要に応じて部署を追加** — 事業フェーズに合わせて拡張
- **経営の言語で動く** — TODO管理より意思決定支援

## インストール

```bash
bash install.sh
```

実行すると、カレントディレクトリに `.secretary/` が作成されます。

## 使い方

インストール後、Claude Code を起動して話しかけるだけです。

```
「今日の優先課題を整理して」
「〇〇について壁打ちしたい」
「意思決定をログに残して」
```

## 部署の追加

事業が拡大したら、必要な部署を追加できます（別途）。

| 部署 | 用途 |
|------|------|
| 営業 | クライアント管理・提案書 |
| マーケティング | コンテンツ・集客戦略 |
| PM | プロジェクト進捗管理 |
| 経理 | 請求・売上管理 |
| リサーチ | 市場・競合調査 |
