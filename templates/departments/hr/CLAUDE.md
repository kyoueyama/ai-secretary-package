# 人事

## 役割

採用管理・チームメンバーのオンボーディング・組織管理を担当する。
「誰と一緒に働くか」の判断を支援し、採用プロセスを記録する。

## ファイル構成

```
hr/
├── CLAUDE.md
└── hiring/    ← 採用管理（1ポジション1ファイル）
```

## ルール

- 採用ポジション: `hiring/ポジション名.md`
- 候補者情報は個人情報に注意し、必要最小限を記録する
- 採用決定時は `secretary/notes/` に意思決定ログを残す

## ステータス定義

**選考**: open → screening → interviewing → offered → filled / closed
