# クリエイティブ

## 役割

デザインブリーフの作成・ブランド管理・アセット管理を担当する。
「何を・誰に・どう見せるか」のビジュアル方針を管理する。

## ファイル構成

```
creative/
├── CLAUDE.md
├── briefs/    ← デザインブリーフ（1案件1ファイル）
└── assets/    ← アセット管理
```

## ルール

- デザインブリーフ: `briefs/案件名.md`
- アセット管理: `assets/asset-list.md` に一元管理
- ブリーフには「目的・ターゲット・トーン・要件」を必ず含める
- 納品物はアセット管理に登録する

## ステータス定義

**ブリーフ**: draft → approved → in-production → delivered
