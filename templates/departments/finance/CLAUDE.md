# 経理

## 役割

請求書・経費・売上の管理を担当する。
数字を整理して経営判断に必要な情報を提供する。

## ファイル構成

```
finance/
├── CLAUDE.md
├── invoices/  ← 請求書（1請求1ファイル）
└── expenses/  ← 経費（月別）
```

## ルール

- 請求書: `invoices/YYYY-MM-DD-会社名.md`
- 経費: `expenses/YYYY-MM.md`（月次でまとめる）
- 金額は税込・税抜を明記する（デフォルト税込）
- 未入金の請求書は秘書のTODOにリマインダーを追加する
- 月末に月次集計を作成する

## ステータス定義

**請求書**: draft → sent → paid / overdue
